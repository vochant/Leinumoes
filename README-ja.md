# Leinumoes

[English](./README.md) | [简体中文](./README-zh_CN.md) | 日本語

Leinumoes は、[MDUI](https://mdui.org/) フロントエンドライブラリと [Node.js](https://nodejs.org/) バックエンドランタイム環境をベースに開発された軽量フォーラムシステムです。ユーザーに簡潔で効率的なオンラインコミュニケーションプラットフォームを提供することを目的としています。このシステムは Markdown と様々な拡張機能を使用して記事を執筆し、シンプルなコミュニティ機能を備えており、小規模な個人フォーラムに適しています。

## 機能

- **ユーザーアカウント管理** ユーザーはアカウントを登録してログインし、情報を編集できます。基本的な権限管理が存在し、管理者の任命または解任、ユーザーの BAN または BAN 解除が可能です。招待制登録を必要に応じて有効にできます。
- **記事広場** ピン留め記事、最新記事の閲覧をサポートし、セクション別に記事を閲覧でき、特定のセクションをお知らせ掲示板として設定できます。セクションは動的な作成と変更をサポートしています。
- **記事執筆** Markdown 構文と様々な拡張機能を使用した記事執筆をサポートしています。記事編集時の画像などの添付ファイルの直接アップロードは現在サポートされていません。
- **記事インタラクション** 記事や他の返信への返信をフロア表示でサポートし、記事への「いいね」または「よくないね」をサポートしています。
- **サイト内メッセージ** ユーザー間のプライベートメッセージ送信またはグループ作成をサポートし、Markdown 構文に対応しています。
- **サイト内通知** 個別ユーザーまたは全ユーザーへのシステム通知送信をサポートし、インタラクション通知機能が組み込まれています。
- **ユーザーアクティビティ** ユーザーが短いタイトルなしのアクティビティを投稿できますが、インタラクション権限は自分、フォロワー、管理者に限定されています。
- **Markdown ツール** Markdown\*Palettes の組み込みプレビュー機能と Leinumoes レンダラーの違いを埋めるため、このツールはサーバーサイドの Leinumoes レンダラーを通じてプレビューできるようにします。
- **クラウドストレージ** ユーザーが添付ファイルをアップロードし、画像構文またはファイル参照構文を使用して挿入することをサポートしています。一般ユーザーには単一ファイルサイズ制限と合計サイズ制限がありますが、管理者には制限がありません。
- **管理バックエンド** 管理者はこのバックエンドを通じてシステムを便利に管理できます。
- **ユーザーサポートセンター** ユーザーが問題に遭遇した場合、サイト内で直接解決できます。
- **静的コンテンツ** サーバーサイドで JSON を使用して静的ページを記述でき、Markdown フラグメント内に HTML を挿入したり、HTML フラグメントを直接使用したりできます。通常の制限を受けません。

## ビルド

このプロジェクトはビルドを必要としません。クローン後、依存関係を補完し、設定ファイルを作成するだけで直接使用できます。

## インストール

このリポジトリをクローンし、プロジェクトディレクトリで以下のインストールコマンドを実行してください。

\*nix システムの場合：

```bash
npm install
mv config.json.in config.json
```

Windows システムの場合：

```batch
npm install
ren config.json.in config.json
```

## 設定

基本設定については、`config.json` を変更してください。フィールドの意味は以下の通りです：

- `title` フォーラムタイトル
- `subtitle` フォーラムサブタイトル
- `key` 暗号化キー
- `port` サーバーリスニングポート
- `announcement` お知らせセクション
- `invite` 招待制

エラーコードとその表示名については、`errids.json` の対応するフィールドを変更してください。

静的コンテンツについては、`static/` ディレクトリに JSON ファイルを作成してください。ファイル名が静的ページのパスになり、形式はそのディレクトリのサンプルファイルを参照してください。

その他のサポートされているコンテンツについては、実行後にブラウザで操作してください。

## 実行

プロジェクトディレクトリで以下のコマンドを実行してサーバーを起動します：

```bash
node index.js
```

サーバー起動後、`http://localhost:<port>` にアクセスしてフォーラムホームページに入ります。`<port>` は `config.json` で設定されたポート番号です。

公開する前に、ユーザーを 1 人登録し、そのパスワードを安全に保管してください。最初に登録されたユーザーは自動的にスーパー管理者になります。通常、サイト全体でスーパー管理者は 1 人のみです。データベース操作なしで任命または解任できる管理者は通常の管理者です。
