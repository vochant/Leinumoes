# Leinumoes 帮助

## 内容格式

1. **我应该如何丰富内容的展现？**
   Leinumoes 系统使用 Markdown 和 KaTeX，您可以参照[格式手册](/docs/format)中规定的格式使用 Markdown 和 KaTeX 的各项功能。同时，如果您希望使用纯文本，请避免触发 Markdown 或 KaTeX 的语法，必要时使用转义。

2. **为什么编辑器和其它地方渲染同一段 Markdown 文本的效果不同？**
   编辑器及其预览使用了[洛谷开发团队](https://github.com/luogu-dev)的 Markdown\*Palettes 项目，而其它地方采用经过定制的 Markdown-It 和 KaTeX 作为渲染引擎。如果您可以修改一个可以专用于 Leinumoes 的 Markdown\*Palettes，则万分感谢。

3. **我应该如何提及其他人？**
   请使用 `@<用户名>`。注意用户名和昵称的区别，用户名是仅能由大小写字母、数字和下划线组成且不为纯数字的唯一标识符。

4. **我应该如何书写公式？**
   在 Leinumoes 中书写公式应当按照 KaTeX 规范。同时，您仅能使用形如 `$ inline $` 或 `$$ display $$` 的格式，而不能使用 `\( inline \)` 或 `\[ display \]` 的格式。

5. **我应该注意什么？**
   推荐在 CJK 字符（如汉字、汉语注音、假名、谚文）与英文、数字、公式间添加一个空格，非句末的半角标点符号之后（左括号则为之前）添加一个空格，不要在全角标点符号前后添加空格。这是一种推荐的留白方式。同时，请善用 Markdown，适当标注重点和添加链接，不要将代码、公式置于代码块、公式块之外。

6. **输入字段有长度限制吗？**
   是的。密码至多为 100 字，其余个人属性（个人简介除外）至多为 50 字，其余至多为 50000 字。

## 用户账号

1. **我能拥有多少个 Leinumoes 账号？**
   通常情况下不超过两个，该范围内无需说明注册意图。如果需要更多，请向站长或管理员提出申请并取得许可，说明注册意图，确保非占用服务器系统资源。同时，如果您需要注册自动化账号或公众账号，无论曾注册几个、将注册几个，都必须向站长或管理员提出申请并取得许可。

2. **我可以注销 Leinumoes 账号吗？**
   暂不支持自助注销，请联系站长或管理员手动注销。请注意，注销不可撤销，且注销后您发布的内容将不会被连带删除，而是显示为无主状态。此时管理员和站长仍能查看其与您的原有 UID 的关联性。

3. **我忘记了密码怎么办？**
   如果您填写了手机号或邮箱，您可以向站长或管理员提出找回密码的申请。如果您从未填写，则除非有其他确凿的证据，否则站长或管理员无法确定您对账号的所有权，将不受理找回密码的请求。如果您在某台设备上仍然处于登录状态，可以提交 Cookie 给站长或管理员作为凭据作为对账号使用权的证明。请注意，该操作将使得站长或管理员直接持有您的原有密码，请三思而后行。

4. **封号后可以做什么？**
   您不能访问大多数页面，但您可以在登出后浏览。如果您有其他账号或在封号时期注册了新账号，当我们认为情况足够可信时有权予以封停。最终解释权归站长所有。如果您需要提出申诉/提交检讨/满期解封/询问细节，应当在登录被封禁账号的状态下进行。

5. **我应该填写个人信息吗？**
   Leinumoes 系统中仅有用户名和密码是必须填写的，数字 ID 是自动分配的。其余信息均为按需填写，请注意一旦填写即对所有人公开，如果不希望公开则不应当填写。

## 社区功能

1. **为什么我无法找到热门评论？**
   Leinumoes 非置顶评论按照时间排序。如果该热门评论未被置顶，则需要找到对应时间。

2. **我如何分享内容？**
   如果您希望使用链接指向原位置，可以直接复制地址，或使用分享按钮复制 Markdown 格式的链接。如果您希望分享或演绎，则应当遵守 [CC BY-NC-SA 4.0](/docs/cc) 协议。

3. **Leinumoes 上的内容免费吗？**
   是的。Leinumoes 本身不包含付费服务，全部我方或用户发布的内容均免费公开，您亦可在遵守 [CC BY-NC-SA 4.0](/docs/cc) 协议的情况下免费分享或演绎任何内容。但是，用户可以在发布的内容中包含指向付费的第三方平台的链接和提示，用户通过 Leinumoes 宣传不属于 Leinumoes 的付费服务在不违反 Leinumoes 规定的情况下不受 Leinumoes 额外限制。同时，站长提供的付款方式仅用于自愿捐赠，不可用于购买任何特权服务。

## 站务问题

1. **什么是管理员制度？**
   Leinumoes 论坛允许除站长（唯一超级管理员）外其他管理员的存在，管理员代表站长维护网站秩序和解决用户问题。以上为管理员职责，管理员应当履行其职责，其判定标准和最终解释权归属 Mirekintoc 所有。管理员并非一成不变，也不公开选拔，亦不与 Leinumoes 构成雇佣关系，为自愿无偿贡献。

2. **网站会关停吗？**
   这取决于 Mirekintoc 的生活和经济状况。在状况良好的时候，尽可能保持开放。同时，作为能力有限的个人，无法保证长期稳定地提供高性能在线服务。
